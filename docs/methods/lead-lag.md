# 手法カード: 業種リードラグ PCA（日米業種リードラグ）

> 参照知識（③・計算なし＝[advisor.md §5](../advisor.md)）。**計算の真実は `quant/lead_lag.py`（テスト済みコード）にあり、このカードはコードの代替ではない**（[ADR-016](../decisions.md)）。AI はこのカードを「いつ・どう使うか」の判断材料として読み、数値は `get_lead_lag()` Tool が返す事実だけを使う（[ADR-014](../decisions.md)）。設計の出所は [ADR-039](../decisions.md)。

## 仮説

日本市場と米国市場は**取引時間帯が非同期**（米国の取引終了後に日本がザラ場を迎える）であり、米国の業種別ショックが**翌営業日の日本の対応業種**にリードラグ（先行・遅行）として波及する。この波及は業種を跨いだ共通因子（市場全体・国スプレッド・シクリカル/ディフェンシブ）に低ランクで乗るため、相関構造を事前部分空間へ正則化した主成分分析（PCA）で安定した低ランク線形予測器として推定できる。

## ユニバース（N=28）

- **US（N_U=11・GICS の SPDR 業種 ETF）**: XLB（素材）・XLE（エネルギー）・XLF（金融）・XLI（資本財）・XLK（情報技術）・XLP（生活必需品）・XLU（公益）・XLV（ヘルスケア）・XLY（一般消費財）・XLC（コミュニケーション）・XLRE（不動産）。配当調整後 close を使用。
- **JP（N_J=17・TOPIX-17 業種別 ETF）**: 1617〜1633。open＋adj_close を使用。

日米の**共通営業日のみ**を使う（inner join）。

## 手順（要約・詳細は `quant/lead_lag.py`）

1. **リターン定義**: PCA 推定は close-to-close `rcc = adjclose_t / adjclose_{t-1} − 1`（日米両方）。戦略/検証は JP の翌日 open-to-close `roc = close_t / open_t − 1`（同日内）。
2. **事前部分空間 V0 ∈ R^{N×K0}**（列直交・Gram-Schmidt）: v1∝1（全体）／v2∝(1_US, −1_JP)（国スプレッド）／v3＝シクリカル(+1)/ディフェンシブ(−1)/0 の符号ベクトル。v2・v3 は順に直交化。
3. **事前エクスポージャー C0**: 長期ベース期間の標準化相関 `Cfull` から `D0=diag(V0ᵀCfullV0)` → `Craw0=V0 D0 V0ᵀ` → `C0=Δ^{-1/2}Craw0Δ^{-1/2}`（対角を 1 に）。
4. **各営業日 t（窓 W_t = {t−L .. t−1}）**:
   1. 窓内で標準化（μ, σ）して相関 `C_t` を作る。
   2. 正則化 `Creg_t = (1−λ)C_t + λC0`。
   3. `eigh(Creg_t)` を降順に取り上位 K 固有ベクトル `V^{(K)}_t` を US/JP ブロックに分割。
   4. US 当日ショックを窓統計で標準化 `z_{U,t}` → 因子スコア `f_t = V^{(K)ᵀ}_{U,t} z_{U,t}` → JP 翌日シグナル `ŝ_{J,t+1} = V^{(K)}_{J,t} f_t`（= `B_t z_{U,t}`, rank ≤ K）。
5. **提示**: 最新の共通営業日基準で算出した業種別シグナルを横断正規化（0..1）し、業種ランキングとして出す。

## パラメータ（モジュール定数・env 不可）

| 記号 | 値 | 意味 |
|---|---|---|
| `L` | 60 | 推定窓（営業日）|
| `K0` | 3 | 事前部分空間の次元 |
| `K` | 3 | 採用する主成分数（rank 上限）|
| `λ` | 0.9 | 事前 C0 への正則化強度（1 に近いほど事前重視）|
| `q` | 0.3 | 検証のロングショート分位（上位/下位 30%）|

## 検証（軽量・提示用途）

履歴で各 t のシグナル `s_{j,t}` と実現 `roc_{j,t+1}` の **Spearman IC**、および 3 分位ロングショート（q=0.3）の **R/R（リスクリワード）・方向的中率**を numpy＋pandas で算出する。Fama-French / Carhart 回帰・フル backtest 基盤は対象外（[ADR-039](../decisions.md)）。
